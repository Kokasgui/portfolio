Este projeto parte de regras etimológicas da gramática portuguesa para criar um gerador de pseudopalavras e um dicionário coletivo de significados atribuídos a estas palavras.

Abrir website Ver código-fonte

Para este projeto, tive a responsabilidade de programar a lógica da geração das pseudopalavras, de acordo com as regras pesquisadas pelos restantes elementos do grupo. Além disso, também fiquei encarregue de programar a interação do website com o servidor, local onde seriam alojados os significados inseridos pelos utilizadores.
